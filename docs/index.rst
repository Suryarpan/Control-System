State Space Analysis
####################

This is the documentation of **State Space Analysis** (libssa). It can simulate various types of systems, represented in State Space format. Currently, support for LTI and LTV systems is in development. This documentation is distributed under GNU FDL v1.3 which is attached with this document in its entirety. The library along with the associated source codes are distributed under GNU GPL v3.
